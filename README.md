# book

使用vue构建的看小说网站

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
pc端请使用移动模式
[在线预览](http://1024cc.cc:3001/)
